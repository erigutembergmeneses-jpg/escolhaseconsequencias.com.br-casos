# Escolhas e Consequências — Casos Complementares

Site complementar ao livro **"Escolhas e Consequências: A Metáfora do Mosaico"** de Erigutemberg Meneses.

## 📚 Sobre o Projeto

Este site hospeda os **22 casos complementares** que foram movidos do manuscrito principal para manter o livro enxuto e didático. São casos de domínio público e composições pedagógicas que ilustram os conceitos de neurociência da decisão, vieses cognitivos e economia comportamental aplicados ao contexto brasileiro e internacional.

## 🧠 Conteúdo

- **22 casos** distribuídos por 12 capítulos do livro
- Casos públicos (documentados em fontes abertas)
- Composições pedagógicas (baseadas em eventos reais com dados anonimizados)
- Casos internacionais de referência

## 🎯 Funcionalidades

- **Filtros por tipo de caso**: Público / Composição Pedagógica / Internacional
- **Busca textual**: Por título, palavra-chave ou lição neural
- **Cards com preview**: Cada caso exibe resumo e lição principal
- **Modal com detalhes**: Conteúdo completo (contexto, bloqueio neural, consequência, lição, o que o cérebro fez)
- **Nota metodológica**: Explicação sobre critérios de anonimização
- **Glossário rápido**: Termos-chave da neurociência da decisão

## 🛠️ Tecnologias

- HTML5 semântico
- CSS3 com variáveis e grid responsivo
- JavaScript vanilla (sem frameworks)
- Font Awesome 6 para ícones
- Google Fonts (Inter)
